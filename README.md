# EasyPHPimagewebsite
A easy PHP Waterfall image website, run in the nginx .| 一个简单的php图片站， 瀑布流样式。
使用起来很简单 目录下有个kim文件夹 读取文件夹下的图片以瀑布流显示，kim文件夹下还可以任意创建子文件夹，子文件夹的名称会显示在导航栏上，点击即可切换图片源至该文件夹下。
